# Arion Privacy Policy

Last Updated: June 2026

## 1. Introduction

Arion is a Discord bot that provides community management, moderation, security, analytics, and engagement features for Discord servers.

This Privacy Policy explains what information Arion collects, how it is used, how it is stored, and the choices available to users and server administrators.

By using Arion in a Discord server, you acknowledge the practices described in this Privacy Policy.

## 2. Information We Collect

Arion may collect and store the following information when necessary to provide bot functionality:

### Discord Identifiers

* User IDs
* Guild (Server) IDs
* Channel IDs
* Role IDs

### Community Management Data

* Moderation records
* Warnings
* Security logs
* Shared ban information
* Reputation and trust scores

### Engagement Data

* XP and leveling information
* Activity statistics
* Achievement progress
* Event participation data
* Inventory and reward data

### Voice Activity Data

* Voice session durations
* Voice activity timestamps

### Configuration Data

* Server settings
* Feature configurations
* Alliance and network settings

## 3. Message Content

Arion processes message content for moderation and security purposes.

Examples include:

* Malicious link detection
* Mention abuse detection
* Capitalization detection
* Moderation command processing

Message content is processed in real time and is not permanently stored by Arion unless required by a moderation action initiated by server administrators.

## 4. How We Use Information

Collected information is used to:

* Provide moderation features
* Detect malicious or disruptive behavior
* Manage security systems
* Maintain leveling and engagement systems
* Generate server analytics
* Provide alliance and networking functionality
* Improve reliability and performance

## 5. Data Storage

Data is stored in databases and application logs controlled by the operators of Arion.

Reasonable efforts are made to protect stored information from unauthorized access.

## 6. Data Sharing

Arion does not sell user information.

Information may be shared only:

* When required to provide alliance or shared moderation features between participating communities.
* When required by law.
* When necessary to protect the security or integrity of the service.

## 7. Artificial Intelligence

Arion does not use collected user data to train artificial intelligence or machine learning models.

## 8. Data Retention

Data is retained only as long as necessary to provide bot functionality, moderation records, security systems, and community features.

Retention periods may vary depending on feature requirements.

## 9. User Rights

Users may contact the bot operator regarding questions about stored data.

Requests may be reviewed and processed where technically feasible and consistent with community moderation requirements.

## 10. Changes to This Policy

This Privacy Policy may be updated from time to time. Continued use of Arion after changes are published constitutes acceptance of the updated policy.

## 11. Contact

For questions regarding this Privacy Policy, contact the operator of Arion through the official support server or project repository.
